# Hands-on Labs – Day 03

## Disaster Recovery, Application Migration, and Azure Management

### Estimated Duration: 4 Hours

## Overview

In this lab, you will implement disaster recovery with Azure Site Recovery, migrate applications and databases to Azure, onboard on-premises servers to Azure Arc, and enable security monitoring. You will start by configuring Azure Site Recovery for Hyper-V VMs, performing test failovers, and executing production failovers. You will then assess legacy applications, set up Azure Migrate for app migration, migrate web apps using App Service Migration Assistant, and migrate databases with Azure Database Migration Service. Finally, you will onboard servers to Azure Arc and enable Microsoft Defender for Cloud, Microsoft Sentinel, and Azure Monitor for comprehensive security and monitoring.

## Objective

In **Day-3** of this workshop you will learn how to:

- Configure Azure Site Recovery for on-premises disaster recovery.
- Perform test failovers and production failovers to Azure.
- Assess and migrate legacy applications to Azure App Service.
- Migrate databases to Azure SQL Database using DMS.
- Onboard on-premises servers to Azure Arc for unified management.
- Enable security and monitoring with Defender for Cloud, Sentinel, and Azure Monitor.

## Prerequisites

Participants should have:

- Familiarity with Azure Site Recovery and disaster recovery concepts.
- Understanding of application migration tools and database migration.
- Access to the lab-provided Azure tenant and on-premises environments.
- Basic knowledge of Azure Arc, App Service, and database services.
- Awareness of security monitoring and compliance in Azure.

## Explanation of Components

- **Azure Site Recovery (ASR)**: Service for disaster recovery by replicating on-premises VMs to Azure.
- **App Service Migration Assistant**: Tool for assessing and migrating web applications to Azure App Service.
- **Azure Database Migration Service (DMS)**: Service for migrating databases to Azure with minimal downtime.
- **Azure Arc**: Platform for managing hybrid and multi-cloud environments from Azure.
- **Microsoft Defender for Cloud**: Unified security management and threat protection for cloud workloads.
- **Microsoft Sentinel**: Cloud-native SIEM for threat detection and response.
- **Azure Monitor**: Comprehensive monitoring solution for applications and infrastructure.
