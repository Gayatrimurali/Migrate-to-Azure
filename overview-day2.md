# Hands-on Labs – Day 02

# Migrating Windows & SQL Server Workloads

### Estimated Duration: 4 Hours

## Overview

In this lab, you will work with Azure Migrate to assess and migrate Windows and SQL Server workloads to Azure. You will start by exploring Azure Migrate projects and built-in assessment tools, evaluate migration readiness using Copilot prompts, set up environments for migration, configure dependency visualization, and perform server migrations. By the end, you will have hands-on experience in discovering on-premises assets, assessing their Azure readiness, and executing migrations to the cloud.

## Objective

In **Day-2** of this workshop you will learn how to:

- Discover and assess on-premises environments using Azure Migrate.
- Evaluate business value and ROI for migrations with Azure Copilot.
- Set up Azure environments for server migration.
- Configure dependency visualization and monitoring agents.
- Perform server migrations using Azure Migrate tools.
- Understand migration planning and execution best practices.

## Prerequisites

Participants should have:

- Familiarity with Azure Migrate and migration concepts.
- Understanding of on-premises server environments and virtualization.
- Access to the lab-provided Azure tenant and on-premises VMs.
- Basic knowledge of assessment tools, dependency mapping, and migration workflows.
- Awareness of Windows and SQL Server workloads in Azure.

## Explanation of Components

- **Azure Migrate**: A centralized service for assessing and migrating on-premises servers, databases, and applications to Azure.
- **Discovery and Assessment**: Tools for discovering on-premises assets and generating Azure readiness reports.
- **Dependency Visualization**: Mapping application dependencies to ensure successful migrations.
- **Azure Copilot**: AI-assisted prompts for evaluating migration business value and ROI.
- **Server Migration**: Tools for replicating and migrating VMs to Azure.
- **Assessment Reports**: Detailed reports on migration readiness, cost estimates, and recommendations.
